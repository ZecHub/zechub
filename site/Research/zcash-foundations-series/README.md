# Zcash Cryptography Foundations Series

A beginner-friendly educational series designed to explain the mathematics, cryptography, and protocol design behind shielded Zcash transactions.

## Learning Path

0. How a Shielded Zcash Transaction Actually Works
1. The Math of "Wrapping Around": Finite Fields
2. Points on a Curve: Elliptic Curves & Group Operations
3. Sealing Envelopes: Hashing & Commitments
4. How the Chain Remembers: Merkle Trees & the Note Commitment Tree
5. Proving Without Revealing: Zero-Knowledge, Intuitively
6. Putting It Together: The Shielded Protocol End-to-End

The series progresses from intuition-first explanations to a complete understanding of how shielded transactions function in Zcash.
