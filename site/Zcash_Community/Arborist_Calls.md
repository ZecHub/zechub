# Arborist Calls

The Zcash Arborist Calls are bi-weekly protocol development meetings focused on tracking upcoming protocol deployment logistics, consensus node implementation issues, and protocol research.

Official site: [https://zfnd.org/arborist-calls/](https://zfnd.org/arborist-calls/)

The calls are open to anyone who wants to contribute to Zcash protocol development or share updates on Zcash node projects.

Register here: [15:00 UTC](https://us06web.zoom.us/webinar/register/WN_Vk7WMz9sRkiIr_hqH_x3LA) / [22:30 UTC](https://us06web.zoom.us/webinar/register/WN_z0k1ipsnRkS4-DGqDhULdA)

The complete list of call notes and agendas can be found in the [arboretum-notes repository](https://github.com/ZcashCommunityGrants/arboretum-notes).
