# Knowledge roadmap

A visual map of ZecHub sorted by experience level. If you are new and not
sure where to start, or you have used Zcash for a while and want the deeper
material, this page points you to the right section of the wiki.

![ZecHub Knowledge Roadmap](PASTE_GITHUB_IMAGE_URL_HERE)

## How to read the map

The roadmap has three levels and two support sections.

**Level 1, onboarding (beginner).** The basics: what Zcash is, a starting
A guide for newcomers and a glossary of terms you will run into.

**Level 2, active use (intermediate).** Choosing a wallet, paying with Zcash
at real merchants, and the privacy tools worth setting up.

**Level 3, power users and developers (advanced).** The technical side: how
zk-SNARKs and the Halo 2 proving system work, running a full node, and where
research happens.

Below the levels, the ecosystem hub covers the organizations and people
behind Zcash, along with the community platforms, and the content roadmap
shows which topics still need pages.

## Who this is for

Newcomers can find a place to start without having to read the whole wiki first.
Contributors can use the map to spot missing or thin topics, then open a
GitHub issue to propose a new page.
