# Filtre de journal des élections

`./filterForElection.sh <logname> <election #>`    

affiche les sighashs et les hauteurs de ballet pour les entrées impliquant le numéro d’élection donné

---

`./numberOfBallets.sh <logname> <election #>`      

affiche le nombre de la dernière entrée (la plus récente) pour la hauteur de ballet correspondant à un numéro d’élection donné
