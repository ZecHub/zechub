# Guide d'utilisation de Keystone Zashi

Guide Twitter :  => [Guide Twitter de l'intégration du portefeuille matériel Zashi x Keystone](https://x.com/zashi_app/status/1869793574880973144) 

Cette intégration marque une évolution significative de l'utilisabilité de Zcash en permettant le stockage à froid de ZEC protégés. La communauté Zcash a déjà connu des revers avec d'autres plateformes de portefeuilles matériels par le passé, mais Keystone s'est imposé comme un partenaire collaboratif prêt à repousser les limites et à innover aux côtés de l'Electric Coin Company. L'équipe Keystone a reçu une subvention de ZCG pour financer sa part du travail.

## Préparation
[Commandez et recevez votre Keystone 3 Pro ou Keystone 3](https://keyst.one) 

Niveau de batterie : assurez-vous que votre appareil keystone a un niveau de batterie supérieur à 20 %.

Câble USB ou carte SD :

- Câble USB pour la mise à jour du firmware (inclus).
- Carte Micro SD (moins de 1 To) pour les mises à niveau (à acheter séparément).

Accès au site officiel de Keystone pour la vérification et la mise à jour du firmware.

Configuration de l'application Zashi sur votre appareil mobile.

## [Guide étape par étape (appareil Keystone)](https://keyst.one/get-started) 


**Choisissez votre langue**
-Vérification de l'appareil (via QR) : la vérification de l'appareil est essentielle pour détecter une éventuelle altération pendant le transport, prévenir les attaques de la chaîne d'approvisionnement et garantir la sécurité du firmware installé.
  - Rendez-vous sur la page de vérification de l'appareil sur le site web de Keystone.
  - Cliquez sur Scan QR Code sur le site officiel.
  - Utilisez l'appareil photo de votre Keystone pour scanner le code QR affiché sur le site web.
  - Un code de vérification apparaîtra sur l'écran de votre Keystone.
  - Saisissez ce code sur le site web pour terminer le processus de vérification.

- **Mise à jour du firmware :**
  - Mise à jour via carte MicroSD
    - Assurez-vous que votre portefeuille Keystone a au moins 20 % de batterie.
    - Insérez la carte SD dans votre ordinateur et formatez-la en FAT32.
    - Téléchargez la dernière version du firmware Cypherpunk depuis la [page de mise à jour du firmware Keystone](https://keyst.one/firmware) et enregistrez le fichier keystone3.bin à la racine de votre carte MicroSD.
    - Placez la carte SD contenant le firmware dans votre portefeuille Keystone.
    - Accédez à l'option "Upgrade" sur votre portefeuille Keystone, puis suivez les instructions à l'écran pour lancer le processus de mise à jour.
  - **Mise à jour via câble USB**
    - Si votre version du firmware est inférieure à 1.0.4, vous devrez effectuer la mise à jour initiale à l'aide d'une carte MicroSD avant de pouvoir procéder aux mises à jour via USB.
    - Assurez-vous que votre portefeuille Keystone a au moins 20 % de batterie.
    - Appuyez sur via USB et utilisez le câble USB pour connecter votre portefeuille Keystone à votre ordinateur. Appuyez sur [Approve] pour autoriser l'accès USB à votre portefeuille Keystone, sinon il pourrait n'autoriser que la recharge.
    - Ouvrez le navigateur web de votre ordinateur et rendez-vous sur la [page de mise à jour du firmware Keystone](https://keyst.one/firmware)
    - Sur la page de mise à jour, cliquez sur le bouton Install Update et suivez les instructions fournies pour installer le dernier firmware.
- **Créer un portefeuille :**
    - Mot de passe sécurisé : choisissez un PIN ou un mot de passe robuste pour protéger votre portefeuille.
    - Nommez votre portefeuille (optionnel) : vous pouvez donner un nom à votre portefeuille pour l'identifier facilement, ou ignorer cette étape.
    - Sélectionnez Create New Wallet si vous configurez un portefeuille pour la première fois.
    - Votre appareil générera une phrase de récupération de 24 mots.
    - Notez cette phrase de récupération et conservez-la en lieu sûr.
    - Confirmez la phrase de récupération en vérifiant les mots dans le bon ordre tel qu'affiché à l'écran.
- **Connecter le portefeuille Zashi + Keystone :**
    - Sur l'appareil Keystone : appuyez sur … sur la page principale
    - Appuyez sur Connect Software Wallet et choisissez Zashi. Le code QR de connexion à Zashi apparaîtra.
    - Dans l'application Zashi : appuyez sur le menu déroulant zashi (en haut à gauche de l'écran)
    - Appuyez sur Connect Hardware Wallet
    - Appuyez sur Ready to Scan
    - Scannez le QR affiché sur l'appareil Keystone
    - Dans l'application Zashi : confirmez le compte du portefeuille Keystone en appuyant sur le compte affiché
    - Appuyez sur Connect en bas de l'écran
