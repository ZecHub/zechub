# Outils d’IA : génération de texte, d’images, de vidéos et d’audio (en ligne)

Voici une liste pour vous simplifier la vie. Savoir comment les utiliser est une autre affaire.

Écrire les meilleurs prompts demande un temps d’apprentissage. Il est possible de maîtriser les bases assez rapidement.

## Génération de texte / Assistants

[Analyseur de prompts](https://novelai.net/tokenizer) (tokenizer) pour apprendre comment fonctionnent les prompts et combien de tokens ils utilisent.

Vous pouvez toujours demander aux assistants chatbot de vous aider à générer un bon prompt.

### Outils gratuits :

- [Claude d’Anthropic](https://claude.ai/) - peut nécessiter un VPN pour s’inscrire (États-Unis uniquement)
- [ChatGPT 3.5](https://chat.openai.com/) - demande un numéro de téléphone
- [Bing Chat (GPT4)](https://www.bing.com/search?q=Bing+AI&showconv=1&FORM=hpcodx) - nombre de prompts limité par jour
- [Llama 2 - 70B](https://www.llama2.ai/) - gratuit sur le site web

### Outils payants :

- [Free2Z Chat2Z](https://free2z.com/ai) - payez avec ZEC / 2Zs pour GPT-3.5 et GPT-4, quelques centimes par prompt
- [Chat GPT-4](https://chat.openai.com/auth/login) - 20 $/mois.

## Génération d’images

### Outils gratuits :

- [ImgnAI](https://imgnai.com/) bot sur Discord ou Telegram - [Guide de démarrage](https://imgnai.gitbook.io/imgnai/) et 
désormais aussi [application web ImgnAI](https://app.imgnai.com/home) avec une connexion à compte séparée 
connexion avec X, Google, Discord, Telegram
- [Bing Dalle3](https://www.bing.com/create) - la version gratuite peut être lente
- [Adobe Firefly](https://www.adobe.com/ee/sensei/generative-ai/firefly.html) - quantité limitée gratuitement
- [Ideogram](https://ideogram.ai/login) - bon pour générer du texte réaliste et lisible dans les images
- [Playground AI](https://playgroundai.com/) - mode gratuit, abonnement à partir de 15 $/mois
- [Clipdrop par stability.ai](https://clipdrop.co/stable-diffusion)
- [StarryAI](https://www.starryai.com) - 5 œuvres/jour
- [Dream par wombo](https://dream.ai/)
- [Craiyon](https://www.craiyon.com/) (clone de DALL-E mini) - abonnement payant à partir de 5 $ pour augmenter la vitesse
- [DALL-E mini](https://huggingface.co/spaces/dalle-mini/dalle-mini) - ancien et basique

### Outils payants :

- [Bot Midjourney](https://discord.com/invite/midjourney) sur Discord - à partir de 10 $/mois - [Guide de démarrage](https://docs.midjourney.com/docs/quick-start)
il est possible d’obtenir environ 100 générations gratuites en évaluant beaucoup d’autres [images ici](https://www.midjourney.com/app/rank-pairs/)
(il faut atteindre le top 2000 des évaluateurs - environ 15 minutes à noter des images)
- [DALL-E 2](https://labs.openai.com/) [](https://openai.com/dall-e-2)- 15 $/115 générations - (15 générations gratuites/mois)
- [DALL-E 3](https://openai.com/dall-e-3/) - inclus dans ChatGPT-4 - 20 $/mois
- [Nightcafe Creator](https://creator.nightcafe.studio/) (4 générations gratuites, forfait payant à partir de 6 $/mois)
