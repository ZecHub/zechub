# Partage de fichiers P2P avec IPFS et le navigateur Brave

Nous pouvons utiliser IPFS pour partager de gros fichiers multimédias entre nous de manière pair-à-pair.

Alors, quel est le problème : les passerelles IPFS ne fonctionnent pas pour les gros fichiers, elles expirent.

**Prérequis** : [IPFS Desktop](https://docs.ipfs.tech/install/ipfs-desktop/) + [navigateur Brave](https://brave.com/)

En bref, la vidéo se trouve à la fin.

Nous pouvons utiliser cela entre nous dans le ZF AV Club pour transférer de gros fichiers multimédias et vous le documenter à tous, parce que c’est amusant et super cool !!

*EN PLUS*, nous collaborerons directement avec l’équipe IPFS de Protocol Labs pour développer ce cas d’usage.
Un autre avantage issu des connexions établies au dWeb Camp !

Sans IPFS installé sur la machine de réception ET en utilisant un navigateur autre que Brave : vous devrez attendre qu’une passerelle charge le fichier. *Cela renverra une erreur 504 pour les gros fichiers.*

Ensuite : comment épingler les fichiers d’autres personnes sur votre machine afin d’éviter le problème du point unique de défaillance lorsque le nœud d’origine se déconnecte.
