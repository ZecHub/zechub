## Zcash.me

![Logo Zcash.me](https://pbs.twimg.com/profile_images/1983177127806533632/LbLuoQjB_400x400.jpg)

### Vue d'ensemble

**Zcash.me** est un annuaire de profils et une plateforme de découverte qui permet à chacun de trouver facilement des utilisateurs Zcash et de leur envoyer des ZEC. Il sert de « Linktree pour Zcash » — une façon simple de partager votre adresse protégée et votre présence sur les réseaux sociaux avec un seul lien.

| Item              | Details |
|-------------------|---------|
| **Website**       | [https://zcash.me/](https://zcash.me/) |
| **Twitter**       | [@zcashme](https://x.com/zcashme) |
| **Profile Format**| `zcash.me/[username]` |
| **Status**        | En ligne (avec des fonctionnalités en cours de développement) |

### Fonctionnalités clés

#### Actuellement disponibles
- **Pages de profil personnalisées** - Page personnalisée avec nom d'utilisateur, biographie, avatar et localisation
- **Affichage d'adresse protégée** - Bouton de copie en un clic pour votre Unified Address
- **Intégration de liens sociaux** - X, LinkedIn, Telegram et sites web personnalisés
- **Système de vérification** - Badge vérifié pour la confiance de la communauté
- **Classement des parrainages** - Suivez les meilleurs parrains et les leaders de la communauté
- **Statut d'activité** - Affiche la dernière heure d'activité

#### Bientôt disponible
| Feature            | Description |
|--------------------|-------------|
| Carte de localisation       | Trouvez des utilisateurs Zcash à proximité |
| Statistiques de la plateforme| Consultez les métriques de croissance |
| Sondages communautaires    | Votez sur les sujets de l'écosystème |
| Actualités et annonces | Mises à jour officielles |
| Forum              | Discussion communautaire |
| Application mobile         | Expérience mobile complète |

### Pour commencer
1. Visitez [https://zcash.me/](https://zcash.me/) et cliquez sur **Join**
2. Choisissez un nom d'utilisateur unique, ajoutez une biographie et une photo de profil
3. Collez votre adresse unifiée Zcash (`u1...`)
4. Connectez vos comptes sociaux
5. Partagez votre lien : `zcash.me/[yourusername]`

## TipZ

![Logo TipZ](https://pbs.twimg.com/profile_images/2023711029805350912/BbXM8cDV_400x400.jpg)

### Vue d'ensemble

**TipZ** est un **protocole de pourboire privé** spécialement conçu pour les créateurs et les cypherpunks. Les fans envoient des pourboires dans n'importe quelle cryptomonnaie majeure ; ils sont automatiquement convertis et livrés sous forme de **ZEC protégés**.

| Item              | Details |
|-------------------|---------|
| **Website**       | [https://tipz.cash/](https://tipz.cash/) |
| **Twitter**       | [@tipz_cash](https://x.com/tipz_cash) |
| **Profile Format**| `tipz.cash/@[username]` |
| **Status**        | En ligne |

> *"tipz.cash est un protocole de pourboire privé pour les créateurs et les cypherpunks."*

### Pourquoi la confidentialité est importante pour les créateurs

#### Liberté financière
- 0 % de frais de plateforme
- Non dépositaire et auto-garde
- Pas d'intermédiaires

#### Conception axée sur la confidentialité
- Chaque pourboire arrive sous forme de ZEC protégés
- Solde du portefeuille privé
- Messagerie chiffrée avec les pourboires
- Option d'anonymat complet pour le créateur

#### Paiements cross-chain
| Currency | Network |
|----------|---------|
| ETH      | Ethereum |
| USDC     | Chaînes multiples |
| SOL      | Solana |
| ZEC      | Zcash direct |

#### Protection des créateurs
- Aucun risque de déplatformisation
- Résistant à la censure
- Aucun KYC requis
