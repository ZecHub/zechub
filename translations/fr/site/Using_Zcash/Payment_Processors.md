# Processeurs de paiement Zcash

## CoinPayments
- **Type de support** : Transparent   
- **Description** : Une passerelle mondiale de paiement en cryptomonnaies qui prend en charge plus de 100 cryptomonnaies, dont Zcash. Offre une intégration simple pour les boutiques en ligne.  
- **URL** : [CoinPayments](https://www.coinpayments.net)  
<img src="https://www.coinpayments.net/2024.03/img/cps-since.svg" alt="Logo CoinPayments" width="200"/>

## CipherPay
- **Type de support** : Shielded   
- **Description** : Acceptez Zcash en quelques minutes, sans dépositaire, sans données acheteur, sans intermédiaire.  
- **URL** : [CipherPay](https://www.cipherpay.app)  
<img src="https://i.ibb.co/3YprqHWG/cipherpay.png" alt="Logo CipherPay" width="200"/>

## Flexa
- **Type de support** : Transparent 
- **Description** : Flexa est un processeur de paiement en cryptomonnaies qui permet aux commerçants d’accepter des actifs numériques, dont Zcash (ZEC), pour les transactions du quotidien.  
- **URL** : [Flexa](https://flexa.co/)
