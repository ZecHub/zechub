# Faucets

<img width="2138" height="878" alt="Capture d’écran_2025-10-30_10-36-45" src="https://github.com/user-attachments/assets/bba0da3b-7064-4680-9535-a0dfcf50cc00" />


Les faucets sont des services qui distribuent gratuitement de petites quantités de cryptomonnaie. Ils sont souvent utilisés par les développeurs pour expérimenter avec les cryptomonnaies et par les utilisateurs pour obtenir de petites quantités de cryptomonnaie sans avoir à en acheter. Ils limitent généralement la quantité de cryptomonnaie qu’un utilisateur peut recevoir pendant une période donnée.
