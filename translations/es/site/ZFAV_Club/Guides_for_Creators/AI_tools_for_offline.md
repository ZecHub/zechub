### Generación de imágenes

¿Dónde descargar modelos para la generación de imágenes? 
En [**HuggingFace**](https://huggingface.co/models?pipeline_tag=text-to-image&sort=trending) y [**Civitai](https://civitai.com/)** hay muchos modelos diferentes.

## Windows - descargar modelos personalizados

Por lo general, se necesita como mínimo una GPU con 4GB de memoria de video para cualquier herramienta de IA.

Las GPU de Nvida funcionan mejor.

- [InvokeAI](https://invoke-ai.github.io/InvokeAI/) (requiere 4GB de memoria GPU)
- [NMKD Stable Diffusion GUI](https://nmkd.itch.io/t2i-gui)
- [EasyDiffusion 3.0](https://github.com/easydiffusion/easydiffusion)

### Mac OS - descargar modelos personalizados

- [Draw Things: AI Generation](https://apps.apple.com/ee/app/draw-things-ai-generation/id6444050820) desde AppStore - gratis
(Puede usar muchos modelos diferentes de StableDiffusion)
- [DiffusionBee](https://diffusionbee.com/) - gratis
(Puede usar muchos modelos diferentes de StableDiffusion)
- [InvokeAI](https://invoke-ai.github.io/InvokeAI/) (requiere 4GB de memoria GPU)

### Linux - descargar modelos personalizados

- [InvokeAI](https://invoke-ai.github.io/InvokeAI/) (requiere 4GB de memoria GPU)
