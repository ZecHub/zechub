# Navegadores web

1. **Brave:** Uno de los navegadores más destacados y seguros del mercado actual. Bloquea anuncios y rastreadores invasivos para la privacidad. Bloquea el almacenamiento de datos de terceros. 

2. **Mulvad:** Un navegador web centrado en la privacidad desarrollado en colaboración entre Mullvad VPN y el Proyecto Tor. Está diseñado para minimizar el rastreo y la toma de huellas digitales.

3. **Tor:** Este navegador bloquea plugins como Flash, RealPlayer, QuickTime y otros: pueden ser manipulados para revelar tu dirección IP. No recomendamos instalar complementos o plugins adicionales en Tor Browser.


## ¿Cuáles son las consideraciones de seguridad al elegir un navegador web?

Al seleccionar un navegador web hay algunas cosas que debes tener en cuenta: las vulnerabilidades y los exploits de seguridad suelen descubrirse con regularidad. Algunos navegadores pueden rastrearse de alguna manera, pero hay navegadores que permiten a los usuarios navegar de forma anónima; [Tor browser](https://www.torproject.org/download/), por ejemplo. 

**Huella digital del navegador**: Si quieres mantener en privado aspectos como el software instalado en tu equipo, la configuración del hardware y más, considera buscar un navegador web que admita funciones de resistencia a la huella digital. 

**Prueba HTTPS**: Otra consideración de seguridad que debes tener en cuenta al seleccionar un navegador web es el protocolo HTTPS Test, que ayuda a cifrar la conexión con el sitio web y oculta cookies, URL y otros tipos de metadatos sensibles. Con HTTPS, la conexión está cifrada y ningún tercero en la red puede leer el contenido que se transfiere entre el servidor y tu navegador. 

**Protección contra cookies de rastreo**: Además, una gran parte de las páginas web de la world wide web tiene rastreadores ocultos de terceros; estos rastreadores leen y escriben cookies en tu navegador. Estas cookies pueden representar amenazas maliciosas porque pueden usarse para rastrear tu navegación entre sitios web; para evitar este rastreo, elige un navegador web que bloquee las cookies de navegación desconocidas.

**Prueba de navegación**: A medida que navegas de una página web a otra, encontrarás muchos enlaces (algunos son enlaces internos, mientras que otros enlazan a otro sitio web). Cuando haces clic en estos enlaces, ciertas APIs del navegador permiten que el primer sitio se comunique con el segundo. Para resolver esto, elige un navegador web que impida que los sitios web compartan datos de rastreo cuando haces clic en un enlace. Al seleccionar ese navegador que sientas que puede ajustarse a tus necesidades, asegúrate de que ofrezca funciones de privacidad sólidas y características esenciales como bloqueadores de anuncios integrados, entre otras. Además, muchos de estos navegadores web admiten extensiones y complementos, y en ocasiones estas extensiones pueden representar riesgos de seguridad; asegúrate siempre de instalar estas extensiones desde un ecosistema seguro o desde fuentes de confianza.
 
## Integración de la wallet de Zcash en Brave Browser

Brave browser está allanando el camino para el ecosistema web3. ¿Cómo? Más de 2 millones de dominios imparables ahora pueden aprovecharse para mostrar sitios web descentralizados. Con las sólidas funciones de seguridad de Brave Browser, las personas pueden explorar internet sin ser rastreadas. Brave y Zcash están uniendo fuerzas para llevar nuevas herramientas de privacidad al ecosistema Web3. Para obtener mayores beneficios, Brave se integraría rápidamente con el protocolo de Zcash mediante la wallet de criptomonedas integrada de Brave para revolucionar las transacciones cripto; esta integración permitiría a los usuarios almacenar, enviar y recibir ZEC de forma segura y muy sencilla, además de otras criptomonedas. La colaboración entre el navegador web Brave y Zcash realmente cambiaría las reglas del juego, ya que los usuarios tendrían un mayor control sobre su actividad en Web3. 

Otro dato intrigante sobre la integración entre Brave browser y Zcash es la función de mensajería privada y descentralizada; Brave, ECC y el equipo de Filecoin Foundation están allanando el camino para una función basada en la privacidad en Brave browser. Esta colaboración tiene como objetivo ofrecer mensajes privados y transmisión de medios a través del protocolo de Zcash. Aquí, Interplanetary File System (IPFS) desempeñará un papel esencial al ofrecer capacidad de almacenamiento seguro para el contenido cifrado.
