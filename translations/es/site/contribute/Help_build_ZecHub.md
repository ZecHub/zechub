# Ayuda a construir ZecHub 

Si eres miembro de la comunidad de Zcash y quieres contribuir a construir ZecHub, hay algunas cosas que son de gran ayuda.

Selecciona enlaces para el boletín y agrégalos a la página de GitHub del boletín: -> [Haz clic aquí para ver las instrucciones](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

Verifica los datos de la documentación de la wiki, sugiere ediciones y propone nuevas páginas en Github

Crea contenido en video en las siguientes categorías:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Produce diseños como carteles/gráficos/animaciones para el ecosistema de Zcash y la privacidad 

Traduce cualquiera de las páginas existentes de la wiki.

Reportar errores

#### Para conocer los detalles sobre los montos de las recompensas, lee nuestra página de contribución -> [aquí](https://zechub.wiki/contribute/contributing-guide#content).
