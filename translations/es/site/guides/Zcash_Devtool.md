# El Devtool de Zcash

[¿Qué es el Zcash-Devtool?](https://github.com/zcash/zcash-devtool?tab=readme-ov-file) 

El Devtool de Zcash es una plataforma para experimentar con Zcash. Está construido por desarrolladores, para desarrolladores, para probar y desarrollar nuevas funcionalidades de Zcash; y no debe considerarse listo para producción. La API de línea de comandos que expone esta herramienta puede cambiará y cambiará en cualquier momento y sin previo aviso. NO comprometas fondos significativos a la gestión de la billetera integrada de zcash-devtool.
