# Usando esta Wiki

Parte disso pode parecer avassaladora. Nós entendemos! Há muita informação aqui. Aqui está um guia simples passo a passo que destaca como você deve navegar pela wiki.

## Novo no Zcash

Se você é novo no Zcash, este é um ótimo lugar para se atualizar. A primeira coisa que você deve fazer é conferir estas páginas:

[Guia para Novos Usuários de Zcash](https://zechub.wiki/start-here/new-user-guide#content)

[Recursos do Zcash](https://zechub.wiki/start-here/zcash-resources#content)

[Fundamentos do Zcash](https://zechub.wiki/start-here/what-is-zec-and-zcash#content)

## Pronto para usar Zcash

Depois de ler os documentos introdutórios e os recursos associados a eles, confira [Usando ZEC](https://zechub.wiki/guides/using-zec-privately#content) e os vídeos de [Tutoriais](https://youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O&si=4vleWUbCGbNJu1-M) para começar a usar ZEC.

## Eu sou um membro da comunidade Zcash

Se você é um membro da comunidade, incrível! A melhor coisa que você pode fazer para se manter atualizado é acompanhar a newsletter semanal [Zcash Ecosystem Digest](https://zechub.substack.com/s/zcash-ecosystem-digest) e seguir todas as contas na [Lista de páginas da comunidade Zcash](https://zechub.wiki/zcash-community/community-links#content).

## Quero contribuir para o ZecHub

Ótimo! Leia [O que é ZecHub](https://zechub.wiki/start-here/what-is-zechub#content) e depois veja [Ajude a Construir o ZecHub](https://zechub.wiki/contribute/help-build-zechub) para aprender como contribuir.
