# A Devtool do Zcash

[O que é o Zcash-Devtool?](https://github.com/zcash/zcash-devtool?tab=readme-ov-file) 

O Zcash Devtool é uma plataforma para desenvolver no Zcash. Ele é construído por desenvolvedores, para desenvolvedores, para testes e desenvolvimento de novas funcionalidades do Zcash; e não deve ser considerado pronto para produção. A API de linha de comando que esta ferramenta expõe pode mudar, e vai mudar, a qualquer momento e sem aviso prévio. NÃO comprometa fundos significativos ao gerenciamento da carteira embutida no zcash-devtool.

Para um guia passo a passo sobre como começar a usar estas ferramentas, consulte [este passo a passo](https://github.com/zcash/zcash-devtool/blob/main/doc/walkthrough.md). Ele documenta um guia completo de como configurar e usar o conjunto de ferramentas do zcash devtool. Destina-se a servir como um guia sobre como fazer a configuração inicial e como adicionar sua própria funcionalidade à ferramenta.


**Avisos de segurança:**
NÃO USE ISTO EM PRODUÇÃO!!!
O aplicativo não foi escrito com foco em segurança. No entanto, ele possui recursos como a criptografia das frases-semente mnemônicas, o que deve torná-lo viável para experimentação em pequena escala, por sua conta e risco.
