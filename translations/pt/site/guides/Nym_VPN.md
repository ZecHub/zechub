# Nym VPN 

Nym é uma mix network (mixnet) em evolução, um tipo de infraestrutura de rede de computadores para privacidade que oculta os metadados do utilizador, separando os endereços IP de origem e de destino. Anonimiza vários tipos de comunicação, incluindo mensagens, transferências de ficheiros, transações de pagamento e navegação web em sites básicos. O projeto é construído com software livre e de código aberto e é descentralizado, mantido por um conjunto distribuído de nós independentes em todo o mundo.
