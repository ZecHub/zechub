# Das Zcash Devtool

[Was ist das Zcash-Devtool?](https://github.com/zcash/zcash-devtool?tab=readme-ov-file) 

Das Zcash Devtool ist eine Plattform zum Hacken an Zcash. Es wurde von Entwicklern für Entwickler gebaut, zum Testen und zur Entwicklung neuer Zcash-Funktionalitäten; und sollte nicht als produktionsreif betrachtet werden. Die Befehlszeilen-API, die dieses Tool bereitstellt, kann sich jederzeit und ohne Vorwarnung ändern und wird sich auch ändern. Setze KEINE erheblichen Geldbeträge für die Verwaltung der im zcash-devtool eingebetteten Wallet ein.
