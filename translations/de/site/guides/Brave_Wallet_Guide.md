# So sendest und empfängst du abgeschirmtes Zcash (ZEC) mit Brave Wallet: Eine Schritt-für-Schritt-Anleitung für Privatsphäre

Brave Wallet ist eine sichere, native Krypto-Wallet, die direkt in den Brave-Browser integriert ist – keine Erweiterungen erforderlich.


## Warum die Unterstützung für abgeschirmtes ZEC wichtig ist


Zcash ist eine der wenigen Kryptowährungen, die standardmäßig Privatsphäre durch abgeschirmte Transaktionen bietet.

Brave Wallet unterstützt jetzt Orchard-abgeschirmtes ZEC und ermöglicht private Transaktionen ohne Software von Drittanbietern.

Nutzer können jetzt ZEC senden/empfangen, ohne Guthaben oder Metadaten offenzulegen – alles direkt im Brave-Browser.

## Teil 1: Brave Wallet einrichten

**Behandelte Schritte:**

1. Öffne den Brave-Browser (vorerst nur auf dem Desktop)

2. Klicke auf das Wallet-Symbol oder gehe zu brave://wallet

3. Richte deine Wallet ein (erstellen oder importieren)

4. Greife auf das Krypto-Dashboard zu


![img1](https://github.com/user-attachments/assets/f54cd1a1-8569-4925-ba1c-7597d030593e)


## Teil 2: Zcash (ZEC) zu Brave Wallet hinzufügen

**Behandelte Schritte:**

1. Klicke im Wallet-Dashboard auf „Assets verwalten“

2. Suche nach ZEC und aktiviere es

3. ZEC erscheint jetzt in deiner Wallet


![img2](https://github.com/user-attachments/assets/6f2b2190-cf55-4394-9d5f-29ff9b5bb525)


## Teil 3: Abgeschirmtes ZEC empfangen (Unified Address)

**Behandelte Schritte:**

1. Klicke in deinen Assets auf ZEC

2. Wähle „Empfangen“

3. Brave erzeugt eine Unified Address (UA) – eine einzige Adresse, die sowohl abgeschirmte als auch transparente Transaktionen unterstützt

4. Kopiere oder scanne den QR-Code

5. Teile deine UA mit dem Absender



![img3](https://github.com/user-attachments/assets/53c940b6-1a03-4fa7-aefa-d3478f678a88)



## Teil 4: Abgeschirmtes ZEC privat senden

**Behandelte Schritte:**

1. Wähle ZEC -> Senden

2. Füge die Unified Address oder abgeschirmte Adresse des Empfängers ein

3. Gib den Betrag ein

4. Bestätige die Transaktionsdetails

5. Klicke auf Senden – die Mittel werden nun privat übertragen, unter Verwendung des Orchard-Pools


![img4](https://raw.githubusercontent.com/Kellyjoe8/zechub/refs/heads/main/Internet_20250808_172118_4.webp)




## Adresstypen

Hier ist eine Erklärung zu jedem Zcash-Adresstyp:

**t-addr (t1...)** – Transparente Adressen funktionieren wie Bitcoin-Adressen, das heißt, Transaktionen sind öffentlich auf der Blockchain einsehbar. Sie sind einfacher mit Nicht-Zcash-Wallets zu verwenden, bieten jedoch keine Privatsphäre.

**z-addr (zs...)** – Abgeschirmte Adressen nutzen die Privatsphäre-Technologie von Zcash (Sprout/Sapling), um Absender, Empfänger und Betrag zu verbergen. Nur Viewing Keys oder der Eigentümer können Details offenlegen.

**UA (u... or QR)** – Unified Addresses kombinieren mehrere Adresstypen (transparent, abgeschirmt, Orchard) in einer einzigen Adresse. Das ermöglicht maximale Kompatibilität, wobei Privatsphäre nach Möglichkeit bevorzugt wird.
