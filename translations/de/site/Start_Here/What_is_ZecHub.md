# Was ist ZecHub?
---

ZecHub ist ein dezentrales Bildungszentrum für Zcash. Das Ziel von [ZecHub](http://zechub.xyz) ist es, eine Bildungsplattform bereitzustellen, auf der Community-Mitglieder gemeinsam Inhalte erstellen, validieren und fördern können, die das Zcash-Ökosystem unterstützen. Das tun wir durch einige Mechanismen.

1. [Tutorials](https://www.youtube.com/playlist?list=PL6_epn0lASLFF28ePj7P5TvtMG_WE7p3O)
2. [Wiki-Dokumentation](https://zechub.wiki/)
3. [Ein Podcast mit Interviews mit Community-Mitgliedern](https://www.youtube.com/playlist?list=PL6_epn0lASLHlNCMtUErX8UfaJK6N9K5O).
4. [Ein wöchentlicher Zcash-Newsletter.](https://zechub.substack.com/)

Die Tutorials, Kurzblogs und der Newsletter sind Open Source und werden auf GitHub gehostet, was bedeutet, dass jede Person aus der Community beitragen kann. So wird sichergestellt, dass die Inhalte korrekt sind und dass der Kanal selbst keinen einzelnen Ausfallpunkt hat.

Wenn du neu bei Zcash bist, nutze [diesen Leitfaden](/site/Start_Here/New_User_Guide).****

---

Im Interesse eines einheitlichen Erscheinungsbilds von ZecHub gibt es eine Vorlage, die du beim Hinzufügen von Seiten zum Repo verwenden kannst.

```markdown
1. All pages should have additional resources (ie. links to z.cash) to support content
2. No marketing or sponsored content
3. No inappropriate content.
```
