# Hilf mit, ZecHub aufzubauen

Wenn du Mitglied der Zcash-Community bist und zum Aufbau von ZecHub beitragen möchtest, gibt es ein paar Dinge, die besonders hilfreich sind.

Kuratiere Links für den Newsletter und füge sie der GitHub-Seite des Newsletters hinzu: -> [Klicke hier für Anweisungen](https://github.com/ZecHub/zechub/blob/main/site/contribute/ZecWeekly_Newsletter.md)

Prüfe die Wiki-Dokumente auf Fakten, schlage Änderungen vor und rege neue Seiten auf Github an

Erstelle Videoinhalte in den folgenden Kategorien:
```
* Zcash explainer videos
* Zcash wallet guide/tutorials
* Third-party application demos
```

Erstelle Designs wie Poster/Grafiken/Animationen für das Zcash- und Privacy-Ökosystem

Übersetze beliebige der bestehenden Wiki-Seiten.

Fehler melden

#### Für Details zu den Bounty-Beträgen lies bitte unsere Mitwirkungsseite -> [hier](https://zechub.wiki/contribute/contributing-guide#content).
