# Processori di pagamento Zcash

## CoinPayments
- **Tipo di supporto**: Transparent   
- **Descrizione**: Un gateway di pagamento in criptovaluta globale che supporta oltre 100 criptovalute, incluso Zcash. Offre una semplice integrazione per i negozi online.  
- **URL**: [CoinPayments](https://www.coinpayments.net)  
<img src="https://www.coinpayments.net/2024.03/img/cps-since.svg" alt="CoinPayments Logo" width="200"/>

---

## CipherPay
- **Tipo di supporto**: Shielded   
- **Descrizione**: Accetta Zcash in pochi minuti, Non-custodial, Zero dati acquirente, Nessun intermediario.  
- **URL**: [CipherPay](https://www.cipherpay.app)  
<img src="https://i.ibb.co/3YprqHWG/cipherpay.png" alt="CipherPay Logo" width="200"/>

---

## Flexa
- **Tipo di supporto**: Transparent 
- **Descrizione**: Flexa è un processore di pagamento in criptovaluta che consente ai commercianti di accettare asset digitali, incluso Zcash (ZEC), per transazioni quotidiane.  
- **URL**: [Flexa](https://flexa.co/)  
<img src="https://framerusercontent.com/images/7D7WwOyaY4PFVW3ZiBzGggx13Xo.svg" alt="Flexa Logo" width="200"/>

## BTCPay Server
- **Tipo di supporto**: Shielded   
- **Descrizione**: BTCPay Server è un processore di pagamento in criptovaluta open-source e auto-ospitato.  
- **URL**: [BTCPay Server](https://github.com/btcpay-zcash/btcpayserver-zcash-plugin)

## ZGo
- **Tipo di supporto**: Transparent e Shielded   
- **Descrizione**: ZGo è una piattaforma di pagamento elettronico che va direttamente dal tuo cliente a te, senza terze parti coinvolte.  
- **URL**: [ZGo](https://zgo.cash/)
