# ZecLedger — ZecHub Hackathon 2026

## Track
Accounting — Reporting, workflows for teams handling ZEC, payment management system.

## What It Does
ZecLedger is the first AI-powered Zcash accounting, reporting and payment management copilot. Built in Rust, connected to Zcash mainnet via Zebra full node and Blockchair API.

## Features
- Accounting: income/expense tracking, net position, fee analysis
- Reporting: CSV and JSON exports, full ledger with running balance
- Payment Management: payment log, pending tracker, confirmation status
- AI Copilot: ask questions in plain English, get research-grade answers
